Detailed Description

Declaration of buffer simulation classes and functions.

This file contains the declarations of various classes and functions for simulating cache and buffer behaviors in a predictive memory system.

The following classes and functions are included:

  • StandardHistoryCacheEntry
  • RealHistoryCacheEntry
  • InfiniteHistoryCache
  • RealHistoryCache
  • HistoryCacheSet
  • Dictionary

DESCRIPTION

This file declares the classes and methods for simulating a predictive memory system. The simulation includes various types of cache entries and caches, including standard history caches, real history caches, and infinite history caches. It also includes a dictionary class for managing deltas and a buffer simulator for simulating access patterns.
