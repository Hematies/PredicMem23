Detailed Description

template<typename T, typename Delta>
class KOrderDFCM< T, Delta >

A class for the K-order DFCM predictor.

This class implements a predictor model using a K-order DFCM (Delta-Feedback Cache Model) with two cache tables. It supports prediction, fitting, and simulation using history-based accesses.

Template Parameters
TThe type of the data (e.g., integer or address).
DeltaThe type of the delta values (e.g., integer).
